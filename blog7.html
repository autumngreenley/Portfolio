Playing around with the API

Getting the key for giphy was fairly straightforward, it's nice that they make it so accessible for people. Playing around with PostMan (or with just the url) it was easy to understand the JSON data and see what was coming from where. There's a lot of options for the gifs (sizes, animated vs nonanimated, etc), which leaves a lot of room for personal choice about what version of a funny animal picture you prefer.

AJAX!

After getting familiarized with the API, I went and set up the Visual Studio project. It's a fairly small project (Only one view, very little formatting) so not a ton was needed on that front. I decided to start off by making a basic AJAX setup to print whatever the user types below the textbox.

After I got that working well, I figured it was time to process that input a little. I figure that, with Javascript, we can add each char to an array (well, a string), then when space is pressed, put that string into another array and empty out the 'single word' array.

From there, I pulled a list of 'important words' from github (a list for verbs, a list for adjectives, and a list for noun) and stored them all in their own arrays. Now, after space is pressed, it checks the entered word against those lists and figures out if it's anything interesting.
